# AI-Powered Research Paper Summarizer and Insights Extractor

## Overview
This project implements a data ingestion pipeline for research papers using PDF, Word, and Excel sources. The data is converted into JSON format, normalized, and processed to extract entities and knowledge triples.

## Features
- Data ingestion from PDF, Word, and Excel
- Unified JSON schema
- Data normalization
- Entity extraction
- Relationship and triple creation
- Agile documentation
